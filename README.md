# VBADailyReport
職業訓練校の日報作成を半自動化するためのVBAソースコード

# 概要
毎日ひな形をコピぺして都度記入の流れが固定化されていたのでその部分を半自動化しました。

# 仕組み
エクセルシート内にマクロボタンを作成し、クリックすることでユーザーフォームを呼び出し、
授業内容などをプルダウンリストから選択し実行するとひな形をある程度編集してくれます。
